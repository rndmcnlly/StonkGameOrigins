# Adam's Terrible Game Engine


## User functions

### setup()
Put your game setup code here. This functions will be called just once when the game starts up.
### update()
Put your per-frame code here. This function will be called repeatedly as long as the browser window is open.

### handleClick(action)
This function will be called with the name of the button that the player just clicked.

 - setTitle(title)
